# CSS (themes)

All "theme" styles can be imported with a single `css` file:

<Story of={CSSThemesStories.CombinedImport} />

This single `theme.css` file uses [prefers-color-scheme](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-color-scheme) media queries to automatically implement a light and dark theme.

It also includes a "manual" theme override (for theme toggling) enabled by adding the appropriate `:root` class.

<Story of={CSSThemesStories.CombinedImportThemeToggling} />
